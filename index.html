TrakStats

A Golf Stat Tracking Application

TrakStats is a golf stat tracking application utilizing the Ruby on Rails framework. I built this application because I saw a need for stat tracking in golf to be more streamlined. Users can enter their stats and monitor their progress. Features include Test Driven Development, User Authentication, Notification Mailers, Video Uploading, Rankings System and an updated Leaderboard. Stripe APIis integrated for online lesson payments.

Cattastic!

Photo Sharing Application

Cattastic! is a photo sharing application built with the Ruby on Rails framework. Posts are loaded in a similar fashion to that of Instagram and users can vote on photos that they like. Features include Test Driven Development, User Authentication, AWS S3 Bucket for photo hosting on Heroku.

The Links Report

Golf Course Review Website

The Links Report is a golf course review website. Modeled after reviews found on the Yelp platform, The Links Report allows users to create accounts so that they can add places and reviews. Features include Google Map API, Star Ratings, Comments, Image Uploading, AWS for photo hosting on Heroku.

CD Chess

An Agile Team Project

CD Chess is a game that was built with Rails as part of an Agile team. Each member had different tasks to complete along the way and we were all responsible for certian additions and features to the project. We held weekly sprints and used Trello to track our progress. Features that I contributed to were joining a game, pawn movement and checking if the game was in stalemate. All code was tested with RSpec and FactoryGirl. Travis CIwas used for continous integration and deployments.
