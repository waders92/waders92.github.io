TrakStats

A golf stat tracking application using Ruby and the Rails framework.

TrakStats is a golf stat tracking application utilizing the Ruby on Rails framework. I built this application because I saw a need for stat tracking in golf to be more streamlined. Users can enter their stats and monitor their progress. Features include Test Driven Development, User Authentication, Notification Mailers, Video Uploading, Rankings System and an updated Leaderboard. Stripe APIis integrated for online lesson payments.

Cattastic!

Photo Sharing Application

Cattastic! is a photo sharing application built with the Ruby on Rails framework. Users can create an account and upload photos of cats (sorry no dogs!). Posts are loaded in a similar fashion to that of Instagram and users can vote on photos that they like. Features include Test Driven Development, User Authentication, Photo Uploading, AWS for photo hosting on Heroku, Acts as Votable gem for "likes".

The Links Report

Golf Course Review Website.

The Links Report is a golf course review website. Modeled after reviews found on the Yelp platform, The Links Report allows users to create accounts so that they can add places and reviews. Features include Google Map API, Star Ratings, Comments, Image Uploading, AWS for photo hosting on Heroku.

Splurty

Random Quote Generator

Splurty was my introduction to Ruby on Rails. It is a one page web application that allows users to add random quotes and the authors that supplied them. Quotes are stored in the database and then dispalyed to the user in a random order. Features include Modal for adding quotes, Database Management, Git and Github for version control, Launched on Heroku.

CD Chess

An Agile Team Project

CD Chess is a chess application that was built with the Ruby on Rails framework utilizing an Agile Team environment. Althought the project is not finished, I feel the experience I took away from the 6 weeks of coding and collaborating was a worthy addition to my portfolio of work. Each member had different tasks to complete along the way and we were all responsible for certian additions and features to the project. We held weekly sprints and used Trello to track our progress. Features that I contributed on were Joining a Game, Pawn Movement and checking if the game is in Stalemate. All code was tested with RSpec and FactoryGirl and Travis CIwas added for continous integration and deployments.
